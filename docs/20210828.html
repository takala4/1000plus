LCP\((M, q)\)に対するメリット関数を考える．\(F(x) = Mx + q\)とする．

Fischerのメリット関数： \[\begin{align} &\Psi(x) = \dfrac{1}{2} \sum_{i} \psi(x_{i}, F_{i}(x))^{2} \\ &\mbox{where} \psi(a,b) = \sqrt{a^{2} + b^{2}} - a - b \end{align}\]

Mangasarian and Solodovのメリット関数： \[\begin{align} M(x) = \left< F(x), x \right> + \dfrac{1}{2\alpha} \left( \left\| [x - \alpha F(x)]_{+} \right\|^{2} - \left\| x \right\|^{2} + \left\| [F(x)- \alpha x]_{+} \right\|^{2} - \left\| F(x) \right\|^{2}\right) \end{align}\]

ここで\(\alpha > 1\)はパラメータ．

次元を\(N=2\)として，ランダムに作成したLCP\((M, q)\)に対して，2つのメリット関数の等高線を描画してみた．

関数が異なるので，関数値そのものや勾配の大きさが変わるのは当たり前だが， 任意の点での凸性・凹性も変わるというのは少し驚き．

もちろん凸っぽい関数の方が扱いは楽なので，問題の性質（特に\(M\)の性質）に応じて， 相性の良いメリット関数を見つける必要があることを改めて認識した．
