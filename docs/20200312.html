TeXで使える適当スタンプを作ってみた．

ブリアンブルに：

\usepackage{fancybox}
\usepackage{color}
\newcommand{\revise}{\textcolor{red}{\ovalbox{\textbf{修正}}}}
\newcommand{\consider}{\textcolor{orange}{\ovalbox{\textbf{検討}}}}
\newcommand{\solved}{\textcolor{blue}{\ovalbox{\textbf{解決}}}}
使い方： 
\begin{itemize}
  \item \consider 検討すること
  \item \revise 修正すること
  \item \solved 解決したこと
\end{itemize}
