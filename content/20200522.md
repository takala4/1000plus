Twitterでみかけて注意しようと思ったこと．


default引数をリストにすると，その変数に変更を加えた場合，呼び出しの度にdefault値が変わってしまう．


```python
def hoge(values=[]):
    values.append('Hi')
    return values
```


この関数を何回か呼び出すと


```
hoge()
>['Hi']

hoge()
>['Hi','Hi']

hoge()
>['Hi','Hi','Hi']
```

となる．頭に入れておこう．というかミュータブルなオブジェクトをデフォルト引数にした
関数を作るのをやめよう．
