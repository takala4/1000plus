正規表現はいつもよく分からない．
特定のTeXコマンドを削除するツールを作るのに，とても時間を要してしまった．

いちおうネスト構造がない場合は，うまく動くロジックは作成できた．


```python
import regex
def delete_command(command, text):
    rexp = regex.compile(r'\\\%s{([^{]*)}'%command)
    return regex.sub(rexp, '\\1' , text)
```
